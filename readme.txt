Page Name: EcoSwap

Description:
The EcoSwap homepage is designed to provide a user-friendly interface for a local sustainable living community. The page includes:

- A header with the site title and navigation links to different sections of the website.
- A hero section with a welcome message and a brief description of the site's purpose.
- Featured listings section displaying eco-friendly items available for swap or share.
- Latest blog posts section showcasing recent articles on sustainable living.
- Upcoming events section listing local sustainability-focused events.
- An aside for community spotlight, highlighting active members and their contributions.
- A footer with links to terms of service, privacy policy, and contact information.

The page is styled using a separate CSS file (styles.css), which ensures a consistent look and feel across the site. The design is responsive and user-friendly, with a focus on readability and ease of navigation.

Deployment:
- Ensure the `index.html` and `styles.css` files are linked and in the same directory.
- Test the site locally by opening the `index.html` file in a web browser.
- Upload both files to GitHub or Canvas as required.

This website serves as a platform to connect community members interested in sustainable living, facilitating the exchange of eco-friendly products and services while promoting environmental awareness.
